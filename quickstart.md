

# 1. Writing and Running Scripts (Quick Start)

Where to write scripts:

Open Blender → go to the Scripting workspace.

Use the Python Console for quick tests.

Use the Text Editor (click New) for full scripts.

How to run scripts:

Write your Python code in the Text Editor.

Press Run Script or Alt + P.

Results appear instantly in the 3D Viewport.

There is an  example script in this repo called create5sphere.py that make 5 spheres in a line




# 2. Adding to Blender with Add-ons 

Why use add-ons?

Avoid switching to the Scripting tab repeatedly.

Turn scripts into buttons that you can click anytime.

There is also addonexample.py in this repo that makes the create5sphere.py into an add-on

# Installing a local add-on

How to install your add-on:

Save your script as a .py file.

Blender → Edit → Preferences → Add-ons.

Click Install… and select your file.

Enable the add-on using the checkbox.

Find your panel in the 3D View sidebar (N key).
